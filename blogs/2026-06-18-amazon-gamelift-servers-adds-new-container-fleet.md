Amazon GameLift Servers now supports customizable Linux capabilities for containers in container group definitions and a new ListContainersNetworkInfo() server SDK action, enabling automatic service discovery and simplified communication between game servers and auxiliary services. The improvements provide finer control over container permissions and access to network information including container names, IDs, and local IP addresses. Available through console, CLI, SDK, and CloudFormation in all AWS regions supporting GameLift except China.
